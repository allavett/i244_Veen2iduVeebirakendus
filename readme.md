# Veenäidu teatamise rakendus

Veebirakenduse eesmärk on lihtsustada veenäidu teatamise protsessi. Rakendust on võimalik kasutada ainult registreerunud kasutajal ja see võimaldab veenäidu teatamist ja tarbimisajaloo jälgimist.
Kasutajad, veenäidud ja tarbimiskohad salvestatakse andmebaasi. Kasutajad on seotud varem sisestatud tarbimiskohtadega - tarbijad neid sisestada ei saa.

## Vaated
### Peamine vaade
Kasutaja saab valida soovitud tegevuse
* Registreerimine
* Login - kui on sisse logitud, muutub logoutiks
* Veenäidu teatamine (Sisselogituna)
* Tarbimisajaloo vaatamine (Sisselogituna)

### Registreerimise vaade 
Kasutaja saab end registreerida, mis loob võimaluse veenäidu teatamiseks ja tarbimisajaloo jälgimiseks
* Maakonna väli - (dropdown valik)
* Linna väli
* Tänava väli
* Maja väli
* Korteri väli
* Kasutajanime väli
* Parooli väli
* Registreerimise nupp

### Login vaade
Kasutaja saab sisse logida
* Kasutajanime väli
* Parooli väli
* Login nupp

### Veenäidu teatamise vaade
Kasutaja saab veenäitu edastada, kui on sisse logitud
* Veenäidu väli
* Saatmise nupp

### Tarbimisajaloo vaade
Kasutaja saab vaadata tarbimisajalugu, kui on sisse logitud
* Perioodi alguse valik
* Perioodi lõpu valik
